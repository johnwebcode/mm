  • by: Techware Solutions
  • email: info@techware.co.in
  • Soulmate Installation - This is probably the simplest way for most users.
    1. Upload the source folder via ftp or file manager
    2. Create a database yourdbname.
    3. Using the link http://yoursitename.com/soulmate/installer/installer.php run the installer
    4. Fill out your database details in the installer and save your changes.
    5. A form for filling out your site details, smtp details will appear, enter your details and save your changes.
    6. Now you will get your site url, admin url, admin password and username

    • 1. Getting Started

    Installing Soulmate

    1. Registered User Features
      1. Easy and user friendly Sign up, Sign In and Account Password Recovery.
      2. Eye catching, user friendly and simple UI/UX.
      3. Can edit and manage user profile details and profile photo.
      4. Only approved profiles and photos will be shown in search and matching profile modules.
      5. Will show your most compatible daily recommendations
      6. Premium member profiles are highlighted and shown in the landing page.
      7. Filters are used for searching your most compatible and perfect match.
    2. Admin Features
      1. Full Administration area to manage all Forums, Support Requests, Admins, Users and Site Settings
      2. Admin only area to manage All Registered Users
      3. Edit & Manage User Details as needed
      4. Create & Manage User Accounts
      5. Have access and can grant access to users to select and load places by default .
      6. Roles can be added and deleted as needed.
      7. Full control over settings used throughout the website
      8. Different payment packages can be added.
    3. Admin Login Details
      1. Login using the following url with username and password admin, admin into the soulmate admin panel.
      2. Username:admin
      3. Password:admin
      4. URL:http://solmate.in/admin
